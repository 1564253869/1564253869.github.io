# 文字RPG对战游戏

这是一个基于Web的文字RPG对战游戏，包含多个具有独特技能的角色。

## 文件结构

- `index.html` - 主页面
- `styles.css` - 样式文件
- `game.js` - 游戏逻辑
- `README.md` - 说明文档

## 如何运行游戏

由于服务器环境限制，您可以通过以下方式在本地运行游戏：

1. 下载所有游戏文件到本地计算机
2. 在文件所在的目录中打开终端或命令提示符
3. 运行以下命令启动一个简单的HTTP服务器：
   
   对于Python 3:
   ```
   python -m http.server 8000
   ```
   
   对于Python 2:
   ```
   python -m SimpleHTTPServer 8000
   ```
   
   或者使用Node.js:
   ```
   npx serve -s . -p 8000
   ```

4. 在浏览器中访问 `http://localhost:8000` 即可开始游戏

## 游戏规则

1. 对战顺序: 1v1，回合交替。召唤物有独立回合，紧随召唤者。主体死亡游戏立刻结束。
2. 技能占用:
   - 未说明=占回合。不占回合技能可组合使用。被动不占回合。
   - 一个回合内只要使用了任意一个占回合技能，本回合就立即结束。
   - 例外：若角色机制明确允许一回合内使用多个占回合技能（如狗的被动），则遵循其特殊机制。
3. 标记与判定:
   - 晕眩：下回合跳过，结束时清标记。
   - 被靠：给目标技能添加"被靠"标记。该标记仅在目标（对手）的下一个回合存在并生效（使用该技能时需判定）。在目标的该回合结束时，无论是否使用、是否剩余，全部清除。
   - 限定技只能用一次。
   - 未说明概率=50%。
   - "视为成功"=强制成功一次判定。
4. 伤害与反弹:
   - 未说明类型需确认。反弹同类型给攻击者。
   - 无普攻。真实伤害无视减伤护盾。
5. 持续时间与叠加:
   - "持续N回合"=当回合生效但不计入，之后N个回合有效。
   - 未说明不可叠加的效果，默认可以叠加。
6. CD规则:
   - N CD = 下N个自身回合内无法使用。
   - 释放当回合不计入，从下一自身回合算起。
   - 空过、额外回合都算CD。
7. 召唤规则:
   - 召唤物紧随召唤者行动。已死召唤物不能重新召唤（同名也不行）。
8. 血量规则:
   - 无血量上限。
9. 模拟对战执行规则:
   - 每一步必须按"最大收益原则"决策，进行量化计算和局势判断。
10. 结束后跳出胜利或失败的大字，并辅以应景的BGM以渲染氛围。

## 角色列表

1. 谭启林 HP: 70
   - 拉雷（不占）：5法伤+晕眩，2 CD
   - 掏一个（占）：16物伤
   - 拔吊毛（不占）：下次攻击必中+150%伤，2 CD

2. 肯追克拉马尔 HP: 80
   - RAP star（不占）：自损15，得1额外回合
   - 靠你娘（占）：15物伤+给目标技能1被靠标记
   - 何意味/何异味（限定，不占，HP<60）：A自损15改一技能自损5；B回20改一技能自损40+得两额外回合

3. 曹博睿 HP: 100
   - 三高人群（被动）：法伤+7反弹一半；50%抵抗控制；物伤+5且自身下次伤害+7
   - 装b（占）：13法伤+自回10+对面下次对自身伤害+5
   - 软硬不吃（占）：持续6回合，>15伤变15，≤15伤取消附加后果

4. 杜找羊 HP: 110
   - 传播HIV（被动：免疫，对手回合开始HIV者受基础真伤；主动占：全场HIV，50% 1倍/50% 4倍基础伤，然后基础伤翻倍）
   - 讲个冷笑话（占）：下回合对方技能50%失效，50%成功但若伤害则反弹40%
   - 包子客（不占，限2次）：下次判定成功

5. 茄子 HP: 80
   - 召唤父亲们（占）：召1~2父，最多3场，死者不复生
   - 画画（占）：复制一个未复制过的主动技能，伤害/治疗减半
   - 废了（限定，占）：对方上回合所有行动失效

6. 狗 HP: 60
   - 妈！滚（占1资源）：全体5伤+下次攻伤-2 (效果可叠加)
   - 跆拳道（占1资源）：8伤+自回3
   - 狗神附体（被动）：
     1. 必定先攻
     2. 30≤HP≤50：每回合2资源 (可在一回合内使用多个占回合技能)
     3. HP<30：每回合3资源 (可在一回合内使用多个占回合技能)，50%免控，攻击必中，二技能回血翻倍(6)
